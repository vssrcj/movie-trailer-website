This is my Project 1 for the Udacity Full Stack Web Developer Nanodegree
------------------------------------------------------------------------

Description
-----------
This simple program show all my favourite movies in a html web page.
If you click on a movie, its trailer is shown.

What's included
---------------
Project_1/
|- entertainment_center.py
|- fresh_tomatoes.py
|- media.py

Requirements
------------
Python is needed. See https://wiki.python.org/moin/BeginnersGuide/Download
Additionally, make sure your operating system opens an .html file 
in a web browser as default.

Running the program
-------------------
Simple run entertainment_center.py with python.
Make sure all the included files are in the same folder.
